Global Electrical Plug & Socket Standards

An expert guide to the 6 most widely used mains wall outlets worldwide—covering pin geometry, earthing protection, and local voltages.

Safety & Compatibility: Wall outlets vary globally by voltage (100V–240V) and pin design. Plugging devices into non-matching voltages without a dual-voltage transformer can cause permanent electrical damage.
Type A • Class II

Type A: North American Un-Grounded

The standard ungrounded utility socket featuring two thin parallel slots. Designed strictly for low-power, double-insulated small electronics and phone chargers across North America and Japan.

Pin Geometry:2 flat parallel blades
Primary Regions:United States, Canada, Japan, Mexico
Mains Voltage:100V – 127V (15 Amp max)
Type B • Class I

Type B: North American Grounded

An upgraded grounded outlet incorporating a U-shaped round earth pin. Provides essential surge protection and grounding for heavy computer hardware, gaming rigs, and kitchen appliances.

Pin Geometry:2 flat blades + 1 round earth pin
Primary Regions:USA, Canada, Mexico, Parts of South America
Safety Profile:Grounded variant of Type A (Fits Type A plugs)
Type C • Europlug

Type C: Universal Europlug Outlet

The universal two-pin ungrounded wall interface found across mainland Europe. Compact and versatile, it accommodates low-power consumer electronics running on high voltage lines.

Pin Geometry:2 round parallel prongs (4mm diameter)
Primary Regions:Europe (excl. UK/Ireland), South America, Asia
Current Limit:Low power only (Rated up to 2.5 Amps)
Type F • Schuko

Type F: German Schuko System

Heavy-duty, recessed round socket fitted with side-earthing spring clips instead of a center pin. Built for robust insulation and high-wattage German and Northern European home machinery.

Pin Geometry:2 round pins + dual side grounding clips
Primary Regions:Germany, Austria, Netherlands, Scandinavia
Power Rating:Heavy-duty grounded (Rated up to 16 Amps)
Type G • BS 1363

Type G: British Standard Safety Outlet

Engineered for maximum safety with child-proof internal shutters that open only when the longer top earth pin enters. Every plug features an internal fuse for local circuit protection.

Pin Geometry:3 rectangular prongs in a triangular layout
Primary Regions:United Kingdom, Ireland, UAE, Singapore, Malaysia
Key Features:Internal fuse (3A–13A) + safety shutters
Type J • SEC 1011

Type J: Swiss Hexagonal Interface

A sleek, space-saving hexagonal socket standard exclusive to Switzerland. It seamlessly accepts standard Type C Europlugs while offering grounded protection through a centralized offset pin.

Pin Geometry:3 round pins (2 power + 1 offset ground pin)
Primary Regions:Switzerland, Liechtenstein
Mains Voltage:230 V / 50 Hz supply systems (10 Amp max)
